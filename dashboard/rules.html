CODE DE CONDUITE

SECTION 01 // OPÉRATIONS ROUTIÈRES

Le respect du code de la route est non-négociable. La conduite dangereuse (arcade, ramming) impactera votre prime de fin de mois.

SECTION 02 // MAINTENANCE FLOTTE

Tout véhicule présentant plus de 5% de dégâts doit être immobilisé pour maintenance immédiate au garage le plus proche.

SECTION 03 // DISCIPLINE RÉSEAU

Le port du tag de l'entreprise et une attitude courtoise sur la radio CB sont obligatoires lors de toutes vos missions officielles.

SECTION 04 // ÉTHIQUE COMMUNAUTAIRE

La VTC Ops repose sur le respect mutuel. Aucun comportement toxique ne sera toléré sur nos outils de communication (Discord/Radio).
