Directive Opérationnelle

CODE DE CONDUITE

SÉCURITÉ ROUTIÈRE // SEC-01

Le respect du code de la route est impératif. Toute conduite jugée dangereuse (ramming, blocage) entraînera une révocation immédiate du permis de transport.

MAINTENANCE VÉHICULE // MAINT-02

Chaque chauffeur est responsable de son unité. Les livraisons doivent être effectuées avec moins de 5% de dégâts pour valider les primes d'excellence.

IDENTITÉ VISUELLE // ID-03

Le port du Tag VTC Ops et les couleurs officielles de la flotte sont obligatoires lors de convoi sous escorte ou de missions publiques.

INTÉGRITÉ DU RÉSEAU // COM-04

La courtoisie sur la Radio CB (Canal 19) et le respect des autres usagers de la route sont le socle de notre réputation européenne.
