Directive Opérationnelle

CODE DE CONDUITE

SÉCURITÉ ROUTIÈRE // SEC-01

Le respect du code de la route est impératif. Toute conduite jugée dangereuse entraînera une révocation immédiate.

MAINTENANCE VÉHICULE // MAINT-02

Les livraisons doivent être effectuées avec moins de 5% de dégâts pour valider les primes d'excellence.

IDENTITÉ VISUELLE // ID-03

Le port du Tag VTC Ops et les couleurs officielles sont obligatoires lors des missions de flotte.

INTÉGRITÉ DU RÉSEAU // COM-04

La courtoisie sur la Radio CB et le respect des usagers sont le socle de notre réputation.
